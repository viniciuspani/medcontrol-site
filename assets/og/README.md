# Imagem Open Graph (compartilhamento em redes sociais)

Coloque aqui o arquivo `og-image.png` usado em todas as meta tags `og:image` e `twitter:image`.

## Especificações

| Item                 | Valor                                                |
|----------------------|------------------------------------------------------|
| Nome do arquivo      | `og-image.png`                                       |
| Dimensão             | **1200 x 630 px** (proporção 1.91:1)                 |
| Formato              | PNG (preferido) ou JPG                               |
| Tamanho do arquivo   | até **300 KB**                                       |
| Área segura          | Mantenha textos a pelo menos 60 px das bordas        |

## Conteúdo sugerido

- Logo MedControl bem visível
- Headline curta: "Nunca mais esqueça um medicamento"
- Subtítulo: "Gestão Inteligente de Medicamentos"
- Fundo com gradiente brand (`#3E7896` → `#7AB89A`)
- Mockup de celular ao lado (opcional)

## Onde é usada

Está referenciada nestas tags em todas as páginas:

```html
<meta property="og:image" content="https://medcontrol.tec.br/assets/og/og-image.png" />
<meta name="twitter:image" content="https://medcontrol.tec.br/assets/og/og-image.png" />
```
